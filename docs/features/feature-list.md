# 功能特性

TimeWrite（智写时光）为小说作者提供完整的写作工作流。

## 书库管理
- ✅ 多作品管理，网格/列表双视图切换
- ✅ 虚拟化滚动，流畅处理大量作品
- ✅ 搜索、排序（时间/字数/书名）
- ✅ 创建/编辑/删除作品，回收站管理（软删除可恢复）
- ✅ 书籍封面设置（JPG/PNG/WebP）
- ✅ 作品信息编辑（书名/作者/简介/封面/每日目标）
- ✅ 每日写作目标 + 进度环可视化

## 章节编辑
- ✅ TipTap 富文本编辑器（H1-H3、加粗/斜体/下划线/字体颜色、图片、表格、代码高亮）
- ✅ 图片插入与缩放，编辑器图片 1200px 等比压缩、封面图片 800px 压缩
- ✅ 图片查看器（放大/缩小/拖拽）+ 图片裁剪对话框（精确选区裁剪）
- ✅ 任务列表、代码块（34 种语言语法高亮）
- ✅ 卷-章节两级目录树（新建/重命名/折叠/状态标签/拖拽排序）
- ✅ 章节标题内联编辑（在编辑器中直接修改）
- ✅ 双保险自动保存（300ms 防抖 + 3 分钟定时）
- ✅ 编辑器状态恢复（上次编辑位置、滚动、光标）
- ✅ 中文字数统计（HTML 解析去标签）
- ✅ Placeholder 占位提示

## 专注写作
- ✅ 专注模式：隐藏侧栏/工具栏/状态栏
- ✅ Esc 一键退出

## 世界观资料库
- ✅ 6 种卡片类型：人物/地点/时间线/势力/物品/其他
- ✅ 搜索、标签、过滤
- ✅ 独立悬浮窗口模式（always_on_top，420×650）

## AI 助手
- ✅ 智谱 BigModel + DeepSeek + 自定义 OpenAI 兼容端点（含 Ollama 本地模型）
- ✅ 推理模型思考过程（Thinking）展示（可折叠）
- ✅ 对话配置与 RAG/Embedding 配置独立解耦
- ✅ 各服务商独立 API Key 管理
- ✅ RAG 上下文检索（向量优先 + SQL LIKE 降级）
- ✅ RAG 连接独立测试 + Embedding 索引状态管理
- ✅ 快捷提示词：续写/润色/剧情推演/角色分析
- ✅ 滑动窗口上下文管理（自动截断保护），对话总结压缩
- ✅ 请求详情面板：查看完整 AI 请求/响应内容
- ✅ AI 工具箱独立悬浮窗口 + 连接状态指示器
- ✅ Agent 技能系统：自动化写作辅助（续写/润色/扩写/分析）

## 版本管理
- ✅ 章节 HTML 内容快照（auto/milestone 类型）
- ✅ 恢复到历史版本
- ✅ 快照预览与删除

## 导入导出
- ✅ 导出为 TXT / Markdown / HTML
- ✅ 导入 TXT，自动按正则识别章节分隔
- ✅ 数据备份：全量/单作品加密备份（AES-256-GCM），支持完整恢复

## 个性化设置
- ✅ 浅色/深色/跟随系统主题切换
- ✅ 护眼模式：暖黄色 / 豆沙绿
- ✅ 全局字体切换（5 种）
- ✅ 字体大小自定义（12-24px）
- ✅ 作品列表网格尺寸（小/中/大）
- ✅ 编辑器显示宽度（移动端/标准/宽屏）

## 插件系统
- ✅ 6 个扩展点（`editor-toolbar` / `editor-sidebar` / `library-card` / `export-format` / `ai-prompt` / `command-palette`）
- ✅ PluginManager 单例驱动
- ✅ 插件安装/启用/禁用/卸载生命周期管理
- ✅ 内置字符统计示例插件

## 深度链接
- ✅ `com.ukcoder.timewrite://` 自定义 URL Scheme
- ✅ 支持外部应用唤起与参数传递

## 独立窗口系统
- ✅ 5 种独立悬浮窗口：世界观资料库 / 版本历史 / 章节总结 / AI 工具箱 / 调试控制台
- ✅ 窗口状态通过 Jotai 原子跨页面共享

## 调试与诊断
- ✅ 调试控制台：SQL 日志广播、Console 拦截、前端日志上报
- ✅ 数据库完整性自动校验
- ✅ 全局 ErrorBoundary 异常捕获与友好提示

## 其他
- ✅ 完整性自动检测脚本
- ✅ 更新器集成（GitHub Releases + GitHub API 双重回退）
- ✅ 代码分割优化（手动 chunk 拆分 TiTap/Icons/State/Router/Markdown/Utils/Virtual）
- ✅ HSL CSS 变量色彩体系（shadcn/ui 兼容语义化色板）
- ✅ 自定义滚动条样式
- ✅ 产品宣传页面（部署至 GitHub Pages）
- ✅ Issue 模板（Bug 报告 + 功能请求，中文表单）
- ✅ 应用版本运行时获取
- ✅ MIT 开源许可证
