# 世界观资料库

世界观资料库是管理小说设定的核心工具，支持 6 种卡片类型。

## 卡片类型

| 类型 | 用途 |
|------|------|
| 👤 人物 | 角色设定、性格、背景故事 |
| 📍 地点 | 场景描述、地理位置 |
| ⏱️ 时间线 | 事件顺序、历史脉络 |
| ⚔️ 势力 | 组织、阵营、帮派 |
| 🎒 物品 | 道具、神器、装备 |
| 📋 其他 | 自由格式设定 |

## 创建卡片

1. 在编辑器右侧打开世界观面板
2. 点击「新建卡片」
3. 选择卡片类型
4. 填写标题和详细内容（富文本）
5. 可选：添加标签
6. 保存

## 管理卡片

### 搜索与过滤
- 按类型过滤：顶部类型标签切换
- 关键词搜索：搜索框实时过滤
- 标签过滤：支持按标签筛选

### 编辑与删除
- 点击卡片进入编辑
- 支持富文本详情（TipTap 编辑器）
- 删除操作不可恢复

## 独立窗口模式

世界观资料库支持**独立悬浮窗口**：

- 点击面板顶部的「弹出窗口」按钮
- 卡片编辑在独立窗口中进行（420×650）
- 窗口始终置顶（`always_on_top`），方便跨窗口参考
- 窗口关闭后数据自动同步到主窗口

## 数据关联

- 世界观卡片绑定到具体作品（`book_id` 外键）
- 每个作品拥有独立的世界观资料库
- 删除作品时，关联的卡片级联删除

## 向量化与 AI 检索

每张卡片包含 `vectorized` 字段，支持接入向量语义检索：
- 在 AI 面板中可一键触发对全部世界观卡片的 Embedding 向量生成
- 向量化后，AI 对话可通过余弦相似度语义检索匹配相关内容
- 配合 RAG 上下文检索，让 AI 助手了解你的完整世界观设定
