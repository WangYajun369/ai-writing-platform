# 章节编辑

TimeWrite 使用 TipTap 富文本编辑器，提供专业的小说创作体验。

## 编辑器功能

### 文本格式化

| 功能 | 快捷键/操作 |
|------|------------|
| H1-H3 标题 | 工具栏选择或 Markdown `#` 语法 |
| **加粗** | `Ctrl/Cmd + B` |
| *斜体* | `Ctrl/Cmd + I` |
| <u>下划线</u> | `Ctrl/Cmd + U` |
| 文字颜色 | 工具栏颜色选择器 |
| 任务列表 | 工具栏复选框按钮 |

### 富媒体支持

- **图片插入**：通过工具栏或拖入图片，支持 JPG/PNG/WebP
- **图片缩放**：拖拽控制点或工具栏调整宽度/高度
- **表格**：插入和编辑表格
- **代码块**：插入代码片段

### 占位提示

编辑器在空白状态显示占位文本，引导开始写作。

## 目录结构

TimeWrite 采用**卷-章节**两级目录：

### 卷管理
- 新建卷：点击目录树顶部的「新建卷」
- 重命名：右键卷名 → 重命名
- 折叠/展开：点击卷名左侧箭头
- 排序：卷按创建顺序排列

### 章节管理
- 新建章节：在卷下点击「+」或右键卷名
- 重命名：双击章节名或右键 → 重命名
- 状态标签：草稿 / 写作中 / 已完成 / 已发布
- 右键菜单：更多操作

## 自动保存

TimeWrite 采用**双保险自动保存**机制：

| 机制 | 触发条件 | 说明 |
|------|---------|------|
| 防抖保存 | 停止输入 300ms 后 | 处理快速连续输入 |
| 定时保存 | 每 3 分钟 | 防止长时间不触发防抖 |

底部状态栏实时显示：
- 当前章节名称
- 字数统计
- 保存状态（保存中 / 已保存 / 最后保存时间）

## 字数统计

- 自动统计中文字数（HTML 解析去标签）
- 同时显示当前章节字数和全书总字数
- 使用 Jotai atom 跨组件实时同步

## 编辑器设置

在设置页面可调整：
- **编辑器宽度**：移动端 / 标准 / 宽屏
- **字体**：衬线 / 黑体 / 宋体 / 楷体 / 微软雅黑
- **字体大小**：12-24px
