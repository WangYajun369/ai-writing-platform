# 书库管理

书库是 TimeWrite 的首页，用于管理所有写作作品。

## 功能概览

- 多作品管理，网格/列表双视图切换
- 虚拟化滚动，流畅处理大量作品
- 搜索、排序（时间/字数/书名）
- 创建/删除作品，书籍封面设置

## 视图切换

书库支持两种视图模式：
- **网格视图**：封面卡片式布局，适合浏览作品
- **列表视图**：紧凑的行式布局，适合快速查找

通过页面顶部的视图切换按钮切换。

## 搜索与排序

### 搜索
在顶部搜索框输入关键字，实时过滤作品列表。

### 排序
支持三种排序方式：
- **按时间**：最近编辑的排在前面
- **按字数**：字数最多的排在前面
- **按书名**：字母/拼音顺序排列

## 创建作品

1. 点击「新建作品」按钮
2. 输入书名（必填）和作者
3. 可选：添加作品描述
4. 点击确认创建

## 封面设置

支持为作品设置自定义封面：

1. 在作品卡片上点击封面区域
2. 选择本地图片文件
3. 支持的格式：JPG、PNG、WebP
4. 封面图片会自动复制到应用数据目录

## 写作进度

### 每日目标
- 在书库中可设置每日写作字数目标
- 进度以环形图可视化展示
- 帮助追踪写作习惯和进度

### 字数统计
每个作品卡片显示当前总字数，实时更新。

## 删除作品

- 右键点击作品卡片
- 选择「删除」选项
- 确认后作品及其所有章节、卷、世界观卡片将被删除
- ⚠️ 删除操作不可撤销
