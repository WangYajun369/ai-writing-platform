# 专注模式

专注模式帮助你屏蔽一切干扰，沉浸在写作之中。

## 进入专注模式

点击编辑器工具栏的**专注模式**按钮进入。

> 注意：专注模式下工具栏会被隐藏，若要再次切换，需先按 `Esc` 退出专注模式。

## 专注模式下的变化

进入专注模式后：

| 隐藏的元素 | 说明 |
|-----------|------|
| 左侧目录树 | 隐藏卷-章节导航 |
| 右侧面板 | 隐藏 AI 助手/版本历史/世界观面板 |
| 工具栏 | 隐藏编辑器顶部工具栏 |
| 底部状态栏 | 隐藏字数/保存状态 |

编辑器会占据整个窗口，提供最大化的写作空间。

## 退出专注模式

- 按 `Esc` 键退出
- 顶部工具栏恢复显示后，可再次点击专注模式按钮

## 状态保持

退出专注模式后，所有面板和布局恢复为进入前的状态，不会丢失任何设置。
