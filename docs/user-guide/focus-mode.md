# 专注模式

专注模式帮助你屏蔽一切干扰，沉浸在写作之中。

## 进入专注模式

点击编辑器工具栏的**专注模式**按钮，或使用以下方式进入：

- 工具栏按钮切换

## 专注模式下的变化

进入专注模式后：

| 隐藏的元素 | 说明 |
|-----------|------|
| 左侧目录树 | 隐藏卷-章节导航 |
| 右侧面板 | 隐藏 AI 助手/版本历史/世界观面板 |
| 工具栏 | 隐藏编辑器工具栏 |
| 底部状态栏 | 隐藏字数/保存状态 |

编辑器会占据整个窗口，提供最大化的写作空间。

## 退出专注模式

- 按 `Esc` 键退出
- 再次点击工具栏专注模式按钮

## 状态保持

退出专注模式后，所有面板和布局恢复为进入前的状态，不会丢失任何设置。
