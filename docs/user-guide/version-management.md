# 版本管理

TimeWrite 提供版本快照功能，保护你的每一次修改。

## 快照类型

| 类型 | 说明 | 触发方式 |
|------|------|---------|
| **自动快照**（auto） | 系统自动创建 | 自动保存时触发 |
| **里程碑快照**（milestone） | 手动创建 | 标记重要版本节点 |

## 创建快照

### 自动快照
自动快照由系统在每次保存时自动创建，无需手动操作。

### 里程碑快照
当你完成一个重要节点（如完成一章、重大修改），可手动创建里程碑快照：

1. 打开右侧版本历史面板
2. 点击「创建快照」
3. 输入快照标签（如"第3章完成"）
4. 保存

## 查看版本历史

在右侧面板切换到**版本历史**标签：
- 列表显示所有快照，按时间倒序
- 每条快照显示类型标签和创建时间
- 里程碑快照显示自定义标签

## 恢复版本

1. 在版本列表中找到目标快照
2. 点击「恢复」按钮
3. 确认恢复操作
4. 当前章节内容将被替换为快照内容

> ⚠️ 恢复操作会覆盖当前内容，建议在恢复前手动创建一个新快照作为备份。

## 删除快照

- 不需要的快照可以删除
- 删除操作不可撤销
- 建议保留里程碑快照作为重要参考

## 存储说明

- 快照保存章节的完整 HTML 内容（`content_html`）
- 存储于 `snapshots` 数据库表
- 章节删除时，关联快照级联删除
