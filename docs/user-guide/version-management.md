# 版本管理

TimeWrite 提供版本快照功能，保护你的每一次修改。

## 快照类型

| 类型 | 说明 | 触发方式 |
|------|------|---------|
| **自动快照**（auto） | 手动创建，不带标签 | 点击「创建快照」不输入标签 |
| **里程碑快照**（milestone） | 手动创建，带自定义标签 | 点击「创建快照」并输入标签 |

## 创建快照

1. 打开右侧版本历史面板
2. 点击「创建快照」
3. （可选）输入快照标签作为里程碑标记（如"第3章完成"）
4. 不输入标签则为自动快照，输入标签则为里程碑快照
5. 保存

## 查看版本历史

在右侧面板切换到**版本历史**标签：
- 列表显示所有快照，按时间倒序
- 每条快照显示类型标签和创建时间
- 里程碑快照显示自定义标签（黄色星标）
- 自动快照显示时钟图标
- 预览按钮可弹窗查看快照的完整 HTML 内容

### 快照预览
- 点击「预览」按钮打开预览弹窗
- 展示快照的完整富文本内容（HTML 渲染）
- 显示创建时间和字数统计
- 点击遮罩或关闭按钮退出预览

## 恢复版本

1. 在版本列表中找到目标快照
2. 点击「恢复」按钮
3. 确认恢复操作
4. 当前章节内容将被替换为快照内容

> ⚠️ 恢复操作会覆盖当前内容，建议在恢复前手动创建一个新快照作为备份。

## 删除快照

- 不需要的快照可以删除
- 删除操作不可撤销
- 建议保留里程碑快照作为重要参考

## 存储说明

- 快照保存章节的完整 HTML 内容（`content_html`）
- 存储于 `snapshots` 数据库表
- 章节删除时，关联快照级联删除
