# 导入导出

TimeWrite 支持多种格式的导入导出，方便作品迁移和分享。

## 导出格式

| 格式 | 说明 |
|------|------|
| **TXT** | 纯文本，适合通用阅读 |
| **Markdown** | 保留标题、加粗等基本格式 |
| **HTML** | 保留完整富文本格式（含图片 base64） |

## 数据备份

除了格式导出，TimeWrite 还提供安全的数据备份功能：

| 备份方式 | 说明 |
|---------|------|
| **全量备份** | 导出所有作品、章节、世界观卡片，AES-256-GCM 加密 |
| **单作品备份** | 仅导出指定作品的完整数据 |

备份文件包含完整数据结构和关联关系，可随时恢复，适合迁移和灾难恢复场景。

> 未来计划（Phase 3）：EPUB、PDF 导出

## 导出操作

1. 在编辑器页面打开目标作品
2. 在工具栏或设置中找到导出功能
3. 选择导出格式（TXT / Markdown / HTML）
4. 选择保存位置
5. 导出完成，含全书所有章节和卷结构

## 导入文本

### 导入 TXT
1. 在书库或编辑器中点击导入
2. 选择 TXT 文件
3. 系统按正则表达式**自动识别章节分隔**
4. 自动创建卷和章节结构，保留段落格式

### 支持的章节分隔格式
- `第X章` 或 `第X节` 格式
- `第 一/二/三… 章` 中文数字格式
- 支持自动识别章节标题作为章节名称

## 数据完整性

- 导出保留 HTML 格式的全部富文本信息
- 导入时保留原始文本段落结构
- 自动生成章节标题
