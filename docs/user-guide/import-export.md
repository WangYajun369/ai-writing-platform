# 导入导出

TimeWrite 支持多种格式的导入导出，方便作品迁移和分享。

## 导出格式

| 格式 | 说明 |
|------|------|
| **TXT** | 纯文本，适合通用阅读 |
| **Markdown** | 保留标题、加粗等基本格式 |
| **HTML** | 保留完整富文本格式 |

> 未来计划（Phase 3）：EPUB、PDF 导出

## 导出操作

1. 在编辑器页面打开目标作品
2. 在工具栏找到导出按钮
3. 选择导出格式
4. 选择保存位置
5. 导出完成

## 导入文本

### 导入 TXT
1. 在书库或编辑器中点击导入
2. 选择 TXT 文件
3. 系统按正则表达式**自动识别章节分隔**
4. 自动创建卷和章节结构

### 支持的章节分隔格式
- `第X章` 或 `第X节` 格式
- 自定义分隔符

## 数据完整性

- 导出保留 HTML 格式的全部富文本信息
- 导入时保留原始文本段落结构
- 自动生成章节标题
