# 个性化设置

TimeWrite 提供丰富的个性化选项，打造专属写作环境。

## 主题系统

### 基础主题

| 主题 | 说明 |
|------|------|
| 🌞 浅色 | 标准亮色主题 |
| 🌙 深色 | 暗色主题，适合夜间写作 |
| 🔄 跟随系统 | 自动跟随操作系统主题设置 |

### 护眼模式
在基础主题上叠加护眼色调：

| 护眼模式 | 说明 |
|---------|------|
| 🌾 暖黄色 | 模拟纸张暖色调，减少蓝光 |
| 🫘 豆沙绿 | 经典护眼绿色调 |

护眼模式与亮色/暗色独立组合，共 **4 套主题**。

## 字体设置

### 全局字体
| 字体 | 风格 |
|------|------|
| Serif（衬线） | 经典书籍字体 |
| Sans（黑体） | 现代简洁风格 |
| Songti（宋体） | 传统中文印刷字体 |
| Kaiti（楷体） | 手写风格 |
| Microsoft YaHei（微软雅黑） | Windows 系统默认 |

### 字体大小
- 范围：12-24px
- 使用 +/- 按钮逐步调整
- 即时预览效果

## 编辑器设置

### 编辑器宽度
| 模式 | 适用场景 |
|------|---------|
| 📱 移动端 | 窄屏预览，适合小窗口或移动端模拟 |
| 📐 标准 | 默认宽度，适合日常写作 |
| 🖥️ 宽屏 | 最大化编辑空间，适合大显示器 |

### 网格尺寸
书库页面作品卡片的显示尺寸：
- **小**：显示更多作品
- **中**：平衡显示
- **大**：突出封面

## 设置持久化

所有个性化设置通过 `localStorage` 持久化保存：
- 重启应用后自动恢复
- 不同操作系统间切换保持设置
