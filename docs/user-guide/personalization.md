# 个性化设置

TimeWrite 提供丰富的个性化选项，打造专属写作环境。

## 主题系统

### 基础主题

| 主题 | 说明 |
|------|------|
| 🌞 浅色 | 标准亮色主题 |
| 🌙 深色 | 暗色主题，适合夜间写作 |
| 🔄 跟随系统 | 自动跟随操作系统主题设置 |

### 护眼模式
在基础主题上叠加护眼色调：

| 护眼模式 | 说明 |
|---------|------|
| 🌾 暖黄色 | 模拟纸张暖色调，减少蓝光 |
| 🫘 豆沙绿 | 经典护眼绿色调 |

浅色和深色主题可与暖黄/豆沙绿护眼模式自由组合，共 **6 种视觉组合**（跟随系统仅支持无护眼/暖黄/豆沙绿）。

## 字体设置

### 全局字体
| 字体 | 风格 |
|------|------|
| 微软雅黑 | 现代清晰，默认字体 |
| 黑体 | 简洁醒目，适合屏幕阅读 |
| 宋体 | 传统中文印刷字体，经典质感 |
| 楷体 | 手写风格，适合文学创作 |

### 字体大小
- 范围：12-24px
- 使用 +/- 按钮逐步调整
- 即时预览效果

## 编辑器设置

### 编辑器宽度
| 模式 | 适用场景 |
|------|---------|
| 📱 移动端 | 窄屏预览，适合小窗口或移动端模拟 |
| 📐 标准 | 默认宽度，适合日常写作 |
| 🖥️ 宽屏 | 最大化编辑空间，适合大显示器 |

### 网格尺寸
书库页面作品卡片的显示尺寸：
- **小**：显示更多作品
- **中**：平衡显示
- **大**：突出封面

## 设置持久化

所有个性化设置通过 `localStorage` 持久化保存：
- 重启应用后自动恢复
- 不同操作系统间切换保持设置
